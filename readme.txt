== Description ==

Simple plug-in for editing the parent comments to any user comments. Use it to edit the threaded structure comments.


== Installation ==

1. Upload the `change-comment-parent` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the Plugins menu in WordPress

Note! This plugin work only if thread comments was enabled at options page.
